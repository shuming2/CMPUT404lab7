
REST: representational state transfer
API
HTTP
URL <=> state, model object

Nouns =>
http://server/mycat
GET     get information =>select
PUT     feed my cat     =>update
DELETE  kill my cat     =>delete
POST    create my cat   =>insert

HTTP: keepalive

CRUD: create(POST) Read update(PUT) delete(DELETE)

XSS: cross-site scripting
    angle brackets are interpreted by browser to tag

CORS: cross-origin resource sharing
    enforced by browser
    XMLHTTPRequest
